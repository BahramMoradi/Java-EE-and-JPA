practical-javaee7-development-wildfly
=====================================

Public repository for code contained in the [Practical Java EE 7 Development using WildFly](http://www.itbuzzpress.com/ebooks/java-ee-7-development-on-wildfly.html)

###### Deploy
```shell
mvn clean install wildfly:deploy
```
###### Test
```shell
http://localhost:8080/javaee7-async-servlet
```
